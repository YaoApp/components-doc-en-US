# Table

Display some structured data on the chart page.

## Example

```json
{
	"name": "Cloud Vendor Overview",
	"width": 12,
	"type": "table",
	"props": {
		"height": 240,
		"columns": [
			{
				"title": "Name",
				"dataIndex": "name"
			},
			{
				"title": "Usage",
				"dataIndex": "usage"
			},
			{
				"title": "Points",
				"dataIndex": "score"
			},
			{
				"title": "Market Share",
				"dataIndex": "rate"
			}
		]
	}
}
```

## Props

```ts
interface IProps {
	height: number
	columns: Array<ColumnsType<any>>
	autoplay?: boolean
	speed?: number
}
```

## API

| Properties | Description  | Type   | Default | Required | Version |
| ---------- | ------------ | ------ | ------- | -------- | ------- |
| height     | height       | string | -       | yes      |         |
| columns    | data columns | string | -       | yes      |         |
| autoplay   | auto scroll  | string | -       | no       |         |
| speed      | scroll speed | string | -       | no       |         |

Column properties see [Antd Table Column](https://ant.design/components/table/#Column)
