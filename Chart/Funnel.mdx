# Funnel

### Used on the chart page

```json
{
	"spend": {
		"bind": "type",
		"link": "/x/Table/type",
		"view": {
			"type": "Funnel",
			"props": {
				"nameKey": "type"
			}
		}
	}
}
```

## Props

```ts
interface IProps {
	data: Array<any>
	nameKey?: string
	height?: number
}
```

## API

| Properties | Description | Type   | Default | Required | Version |

| height     | height      | number | -       | no       |         |
