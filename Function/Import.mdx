# Import

Data import for importing Excel, CSV files.

## Example

<Notice type='success'>
	Note: Import is only valid when defined in the JSON of the table page, and data cleaning is
	supported during data import.
</Notice>

tables/[*].json `list.actions`

```json
{
	"actions": {
		"import": {
			"props": {
				"api": {
					"setting": "/api/xiang/import/order/setting",
					"mapping": "/api/xiang/import/order/mapping",
					"mapping_setting": "/api/xiang/import/order/mapping/setting",
					"preview": "/api/xiang/import/order/data",
					"preview_setting": "/api/xiang/import/order/data/setting",
					"import": "/api/xiang/import/order"
				}
			}
		}
	}
}
```

## Props

```ts
export interface IProps {
	api: {
		setting: string
		mapping: string
		mapping_setting: string
		preview: string
		preview_setting: string
		import: string
	}
}
```

## API

| Properties | Description | Type        | Default | Required | Version |
| ---------- | ----------- | ----------- | ------- | -------- | ------- |
| api        | interface   | [api](#api) | -       | yes      |         |

### api

| Properties      | Description                         | Type   | Required |
| --------------- | ----------------------------------- | ------ | -------- |
| setting         | setting interface                   | string | yes      |
| mapping         | field mapping to get data interface | string | yes      |
| mapping_setting | field mapping setting interface     | string | yes      |
| preview         | data preview data interface         | string | yes      |
| preview_setting | Data preview setting interface      | string | yes      |
| import          | import interface                    | string | yes      |
