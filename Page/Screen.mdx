# Screen

## Scenes

- Display key metrics

- Real-time data monitoring

## Props

<Notice type='success'>
	The large screen configuration item is the same as Chart, the difference is that the large
	screen does not have a filter bar, and the large screen has an additional top indicator bar,
	and the layout of the large screen is a grid layout, and the vertical layout needs to be set
	at the same time. width and height, width is 24 grids, height is 12 grids.
</Notice>

```ts
interface IProps {
	page: {
		// top indicator bar data
		option: {
			left_index: [
				'Total output value of cloud computing industry',
				'Cloud computing related companies',
				'Licensed companies',
				'China cloud computing share'
			]
			right_index: [
				'Industry Monthly Growth Rate',
				'Industry Quarterly Growth Rate',
				'Industry Annual Growth Rate',
				'Market Demand Growth Rate'
			]
		}
	}
}
```
