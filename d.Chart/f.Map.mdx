# Map

The map component, usually used in the center of the large screen, is used to display some real-time geographic location-related data.

## Scenes

- Population migration trend during Spring Festival.

- Trends in the movement of people in domestic air traffic.

## Example

```json
{
	"name": "Cloud vendor distribution",
	"width": 12,
	"height": 8,
	"type": "map",
	"props": {
		"series": [
			{
				"name": "location",
				"type": "map",
				"map": "china"
			}
		]
	}
}
```

## Props

```ts
export interface IProps {
	name: string
	height: number
}
```

## API

| Properties | Description | Type   | Default | Required | Version |
| ---------- | ----------- | ------ | ------- | -------- | ------- |
| name       | data key    | string | -       | yes      |         |
| height     | height      | string | -       | yes      |         |

For more properties, see [Echarts Map](https://echarts.apache.org/examples/en/index.html#chart-type-map)
