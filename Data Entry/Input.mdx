# Input

Entering content through the mouse or keyboard is the most basic form field wrapper.

## Scenes

- When the user is required to enter form field content.

- Provide combination input box, input box with search, and size selection.

## Example

```json
{
	"Input": {
		"label": "Input",
		"edit": {
			"type": "input",
			"props": {
				"value": ":group",
				"placeholder": "Please enter a grouping"
			}
		}
	}
}
```

## Props

```ts
interface IProps extends InputProps {
	value: string
}
```

## API

| Properties | Description | Type   | Default | Required | Version |
| ---------- | ----------- | ------ | ------- | -------- | ------- |
| value      | bind value  | string | -       | yes      |         |

For other properties, see [Antd Input](https://ant.design/components/input/)
