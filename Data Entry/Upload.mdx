# Upload

File selection upload and drag and drop upload controls.

## Scenes

Uploading is the process of publishing information (web pages, text, pictures, videos, etc.) to a remote server through web pages or uploading tools.

- When one or several files need to be uploaded.

- When you need to use drag and drop interaction.

## Example

```json
{
	"Upload": {
		"label": "Upload",
		"edit": {
			"type": "upload",
			"props": {
				"value": ":images",
				"filetype": "image",
				"multiple": true,
				"maxCount": 1
			}
		}
	}
}
```

## Props

```ts
interface IProps extends UploadProps {
	value: Array<string>
	filetype: 'image' | 'file' | 'video'
	desc?: string
}
```

## API

| Properties | Description        | Type   | Default | Required | Version |
| ---------- | ------------------ | ------ | ------- | -------- | ------- |
| value      | bind value         | string | -       | yes      |         |
| filetype   | IProps['filetype'] | string | -       | yes      |         |
| desc       | text description   | string | -       | no       |         |

For other properties, see [Antd Upload](https://ant.design/components/upload/)
