# Item

Item is the parent component of all data entry components, and Item handles form components uniformly through event hijacking.

## Scenes

<Notice type='warning'>
	Note: This component cannot be used alone, the data entry type component will have its own
	Item.
</Notice>

## Example

```json
{
	"Item": {
		"label": "Item",
		"edit": {
			"type": "input",
			"props": {
				"value": ":group",
				"rules": [
					{
						"required": true
					}
				]
			}
		}
	}
}
```

## Props

```ts
interface IProps extends FormItemProps {
	hide_label?: boolean
}
```

## API

| Properties | Description           | Type    | Default | Required | Version |
| ---------- | --------------------- | ------- | ------- | -------- | ------- |
| hide_label | whether to show label | boolean | -       | no       |         |

For other properties, see [Antd Form.Item](https://ant.design/components/form/#Form.Item)
