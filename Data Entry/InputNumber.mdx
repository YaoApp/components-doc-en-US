# InputNumber

With the mouse or keyboard, enter a numerical value within the range.

## Scenes

- When it is necessary to obtain standard values.

- Provides formatted numeric input to constrain the input.

## Example

```json
{
	"InputNumber": {
		"label": "InputNumber",
		"edit": {
			"type": "inputNumber",
			"props": {
				"value": ":amount",
				"placeholder": "Please enter the amount"
			}
		}
	}
}
```

## Props

```ts
interface IProps extends InputNumberProps {
	value: string
}
```

## API

| Properties | Description | Type   | Default | Required | Version |
| ---------- | ----------- | ------ | ------- | -------- | ------- |
| value      | bind value  | string | -       | yes      |         |

For other properties, see [Antd InputNumber](https://ant.design/components/input-number/)
