# Cascader

Cascading selection boxes.

## Scenes

- Need to select from a set of related data sets, such as provinces, municipalities, company levels, classification of things, etc.

- When selecting from a larger data set, separate with multi-level classification for easy selection.

- Compared with the Select component, the selection can be completed in the same floating layer, which has a better experience.

## Example

```json
{
	"Cascader": {
		"label": "Cascader",
		"edit": {
			"type": "cascader",
			"props": {
				"value": ":cascader",
				"showSearch": true,
				"placeholder": "Please select the manufacturer",
				"remote": {
					"api": "/api/cascader/search",
					"query": {
						"select": ["id", "name"]
					}
				}
			}
		}
	}
}
```

## Props

```ts
interface Option {
	value: string | number
	label?: React.ReactNode
	disabled?: boolean
	children?: Array<Option>
}

interface IProps extends TreeProps {
	options: array<Option>
	remote: {
		api: string
		query: {
			select: Array<string>
		}
	}
}
```

## API

| Properties | Description            | Type              | Default | Required | Version |
| ---------- | ---------------------- | ----------------- | ------- | -------- | ------- |
| value      | bind value             | string            | -       | yes      |         |
| options    | list of options        | IProps['options'] | -       | no       |         |
| remote     | list of remote options | [Remote](#Remote) | -       | no       |         |

### Remote

| Properties | Description | Type                      | Required |
| ---------- | ----------- | ------------------------- | -------- |
| api        | interface   | string                    | yes      |
| query      | interface   | IProps['remote']['query'] | no       |

For other properties, see [Antd Cascader](https://ant.design/components/cascader/)
