# Mentions

Mention, usually used to associate or notify someone.

## Scenes

- When a notification needs to be sent to one or more people in a chat application.

- In the background system, when the current data needs to be notified or associated with one or several objects.

## Example

```json
{
	"Mentions": {
		"label": "Mentions",
		"edit": {
			"type": "mentions",
			"props": {
				"value": ":manu_id",
				"autoSize": true,
				"search": {
					"api": "/api/select/search",
					"key": "keyword"
				}
			}
		}
	}
}
```

## Props

<Notice type='success'>Mentions component has the same API as Select component. </Notice>

```ts
interface IProps extends MentionProps {
	value: string
	string?: '1' | '0' | undefined
	options?: Array<{ label: string; value: string | number }>
	remote?: {
		api: string
		query: {
			// Whether to use value as query parameter
			useValue?: boolean
			// select the fields to return
			select?: Array<string>
		}
	}
	search?: {
		api: string
		key: string
	}
}
```

## API

| Properties | Description                        | Type                     | Default | Required | Version |
| ---------- | ---------------------------------- | ------------------------ | ------- | -------- | ------- |
| value      | bind value                         | string                   | -       | yes      |         |
| string     | whether to convert value to string | string                   | -       | no       |         |
| options    | list of options                    | Array<[Option](#Option)> | -       | no       |         |
| remote     | list of remote options             | [Remote](#Remote)        | -       | no       |         |
| search     | remote search                      | [Search](#Search)        | -       | no       |         |

### Option

| Properties | Description | Type             | Required |
| ---------- | ----------- | ---------------- | -------- |
| label      | text        | string           | yes      |
| value      | value       | string \| number | yes      |

### Remote

| Properties | Description | Type                | Required |
| ---------- | ----------- | ------------------- | -------- |
| api        | interface   | string              | yes      |
| query      | interface   | IProps.remote.query | no       |

### Search

| Properties | Description         | Type   | Required |
| ---------- | ------------------- | ------ | -------- |
| api        | interface           | string | yes      |
| key        | query parameter key | string | yes      |

For other properties, see [Antd Mentions](https://ant.design/components/mentions/)
