# CheckboxGroup 

Group of checkboxes.

## Scenes

- When making multiple selections from a set of options.

- Instead of the Select component, display options directly.

## Example

```json
{
	"CheckboxGroup": {
		"label": "CheckboxGroup",
		"edit": {
			"type": "checkboxGroup",
			"props": {
				"value": ":checkboxGroup",
				"options": [
					{
						"label": "Apple",
						"value": 1
					},
					{
						"label": "Pear",
						"value": 2
					},
					{
						"label": "Orange",
						"value": 3
					}
				]
			}
		}
	}
}
```

## Props

```ts
interface Option {
	label: string
	value: string
	disabled?: boolean
}

interface IProps extends TreeProps {
	options: array<Option>
	remote: {
		api: string
		query: {
			select: Array<string>
		}
	}
}
```

## API

| Properties | Description            | Type              | Default | Required | Version |
| ---------- | ---------------------- | ----------------- | ------- | -------- | ------- |
| value      | bind value             | string            | -       | yes      |         |
| options    | list of options        | IProps['options'] | -       | no       |         |
| remote     | list of remote options | [Remote](#Remote) | -       | no       |         |

### Remote

| Properties | Description | Type                      | Required |
| ---------- | ----------- | ------------------------- | -------- |
| api        | interface   | string                    | yes      |
| query      | interface   | IProps['remote']['query'] | no       |

For other properties, see [Antd CheckboxGroup](https://ant.design/components/checkbox/#Checkbox-Group)
