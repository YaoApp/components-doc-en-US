# Schedule

A schedule is usually used to enter or present some schedule-related information.

## Scenes

- Daily consumption recipes.

- List of daily activities.

## Example

<Notice type='warning'>Note: The schedule needs to be used with specific settings. </Notice>

```json
{
	"Schedule": {
		"label": "Schedule",
		"edit": {
			"type": "schedule",
			"props": {
				"value": ":schedule",
				"api": "/api/xiang/setting/schedule",
				"id": ":id"
			}
		}
	}
}
```

## Props

```ts
interface IProps extends InputProps {
	value: string
	api: string
	id: string
}
```

## API

| Properties | Description              | Type   | Default | Required | Version |
| ---------- | ------------------------ | ------ | ------- | -------- | ------- |
| value      | bind value               | string | -       | yes      |         |
| api        | setting interface        | string | -       | yes      |         |
| id         | associated form variable | string | -       | is       |         |
