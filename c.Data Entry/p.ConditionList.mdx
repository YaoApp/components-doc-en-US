# ConditionList

Condition list, supports CURD operation on the specified type of data list in Form and supports linkage.

## Example

<Notice type='warning'>Note: The condition list needs to be used with a specific setting. </Notice>

```json
{
	"ConditionList": {
		"label": "ConditionList",
		"edit": {
			"type": "conditionList",
			"props": {
				"setting": "survey",
				"value": ":survey",
				"query": {
					"sort": "$index"
				}
			}
		}
	}
}
```

## Props

```ts
interface IProps extends InputProps {
	value: string
	setting: string
	query?: { [key: string]: string }
}
```

## API

| Properties | Description                                                                        | Type            | Default | Required | Version |
| ---------- | ---------------------------------------------------------------------------------- | --------------- | ------- | -------- | ------- |
| value      | bind value                                                                         | string          | -       | yes      |         |
| setting    | setting name                                                                       | string          | -       | yes      |         |
| query      | Submit parameter, which will be integrated into data as it is when submitting data | IProps['query'] | -       | No       |         |
