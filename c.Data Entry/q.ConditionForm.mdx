# ConditionForm

Condition list form, consuming structured data produced by ConditionList.

## Scenes

- Fill in the daily food recipes.

- Completion of daily activity list.

## Example

<Notice type='warning'>
	Note: The setting of the condition list form needs to be obtained by writing a specific JSON
	flow.
</Notice>

```json
{
	"ConditionForm": {
		"label": "ConditionForm",
		"edit": {
			"type": "conditionForm",
			"props": {
				"value": ":survey_result",
				"api": "/api/survey/setting/survey_result",
				"id": ":id"
			}
		}
	}
}
```

## Props

```ts
interface IProps extends InputProps {
	value: string
	api: string
	id: string
}
```

## API

| Properties | Description              | Type   | Default | Required | Version |
| ---------- | ------------------------ | ------ | ------- | -------- | ------- |
| value      | bind value               | string | -       | yes      |         |
| api        | setting interface        | string | -       | yes      |         |
| id         | associated form variable | string | -       | is       |         |
