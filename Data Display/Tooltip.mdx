# Tooltip

Prompt, the description text appears when the mouse hovers, which is often used to simplify the page and focus the content.

## Scenes

- menu item name

- Detailed description of the object

## Example

```json
{
	"instruction": {
		"label": "Description",
		"view": {
			"type": "tooltip",
			"props": {
				"value": ":desc"
			}
		}
	}
}
```

## Props

```ts
type IProps = TooltipProps & {
	value: string
}
```

## API

| Properties | Description          | Type   | Default | Required | Version |
| ---------- | -------------------- | ------ | ------- | -------- | ------- |
| value      | detailed description | string | -       | yes      |         |

For other properties, see [Antd Tooltip](https://ant.design/components/tooltip/)
