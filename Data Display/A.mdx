# A

Click the link to jump to the external site.

## Scenes to be used

- Jump to third-party sites

- Jump to the page of the custom extension

- Jump between systems

## Example

<Notice type='warning'>
	Note: As an extension of the a tag, the target attribute of this component is "_blank" and cannot
	be changed.
</Notice>

```json
{
	"Link": {
		"bind": "link",
		"view": {
			"type": "A",
			"props": {}
		}
	}
}
```

## Props

```ts
interface IProps
	extends React.DetailedHTMLProps<
		React.AnchorHTMLAttributes<HTMLAnchorElement>,
		HTMLAnchorElement
	> {}
```
