# A

Click the link to jump to the external site.

## Scenes

- Jump to third-party sites

- Jump to the page of the custom extension

- Jump between systems

## Example

<Notice type='warning'>
	Note: As an extension of the a tag, the target attribute of this component is "_blank" and
	cannot be changed.
</Notice>

```json
{
	"Link": {
		"label": "External link",
		"view": {
			"type": "a",
			"props": {
				"value": ":link"
			}
		}
	}
}
```

## Props

```ts
interface IProps
	extends React.DetailedHTMLProps<
		React.AnchorHTMLAttributes<HTMLAnchorElement>,
		HTMLAnchorElement
	> {
	value: string | undefined
}
```

## API

| Properties | Description | Type   | Default | Required | Version |
| ---------- | ----------- | ------ | ------- | -------- | ------- |
| value      | text        | string | -       | yes      |         |
| href       | link        | string | -       | no       |         |
