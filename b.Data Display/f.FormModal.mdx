# FormModal

Form pop-up window, allowing the content of the form page to be displayed in the pop-up window.

## Scenes

- Use on Table for quick actions

- Calling editing functions of another module across modules

## Example

```json
{
	"source name": {
		"label": "Source",
		"view": {
			"type": "formModal",
			"props": {
				"type": "view",
				"value": ":channel.name",
				"formId": ":channel.id",
				"formName": "channel"
			}
		}
	}
}
```

## Props

```ts
interface IProps {
	type?: 'view' | 'edit'
	value: string
	formId: string
	formName: string
}
```

## API

| Properties | Description         | Type             | Default | Required | Version |
| ---------- | ------------------- | ---------------- | ------- | -------- | ------- |
| type       | form type           | 'view' \| 'edit' | -       | no       |         |
| value      | text                | string           | -       | yes      |         |
| formId     | form association ID | string           | -       | yes      |         |
| formName   | form name           | string           | -       | yes      |         |
