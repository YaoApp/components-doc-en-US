# InputNumber

With the mouse or keyboard, enter a numerical value within the range.

## Scenes to be used

- When it is necessary to obtain standard values.

- Provides formatted numeric input to constrain the input.

## Example

```json
{
	"InputNumber": {
		"bind": "InputNumber_value",
		"edit": {
			"props": {}
		}
	}
}
```
