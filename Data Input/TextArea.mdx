# TextArea

TextArea, which supports the input of a fairly long, unrestricted text.

## Scenes to be used

- Site reviews.

- Some detailed descriptions, such as user address, etc.

- A comment in the feedback form.

## Example

```json
{
	"TextArea": {
		"bind": "TextArea_value",
		"edit": {
			"type": "TextArea",
			"props": {
				"rows": 3
			}
		}
	}
}
```
